FFTF - Fonts

Montserrat

Use mainly for unique headliness and certain buttons like our links in the footer of the website.

http://www.google.com/webfonts/specimen/Montserrat

Oleo Script

Sub heading type used to highlight sections on a page such as the projects on the FFTF home page or the column title in sub-pages

http://www.google.com/webfonts/specimen/Oleo+Script

Helvetica Bold

Main buttons text. used on the donation, sign up and act now buttons at 18pt size

Default type, if not available then Arial or any san-serif should be used

Helvetica (TT)

Main body text. Used for all body text situations. Use in bold for headlines related to that text.

Default type, if not available then Arial or any san-serif should be used

FFTF - Colors

Red #EE4741

Accent color, must be used in contextual terms. Use with the heart symbol or the flame icon on active projects.

Yellow #FBC337

Primary color for showing active and important things. Must be used carefully to enhance the brand of Fight for the Future.

Green #19A96A

Green must be used strictly for hyperlinks. Hover state for all hyperlinks must be the darker shade of green #026600

Gray #202020

Background color on home page. This color must be used only as a background color and in strict context.

White #FFFFFF

White must be used as a background image in all subpages for easier readability and consistency.

Blue #4965A8

Blue color here is derived from the main image we have. This color must be only used to overlay images that reside below the FFTF logo
